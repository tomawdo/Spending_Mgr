Funzionalità principali:

- Aggiungere una spesa:
        - Inserire la data, la categoria (es. cibo, trasporti, intrattenimento) e l’importo.
- Visualizzare il riepilogo delle spese:
        - Mostrare tutte le spese registrate in formato tabellare.
- Calcolare le spese totali:
        - Sommare le spese totali e, eventualmente, filtrare per una specifica categoria o periodo.
- Salvare e caricare i dati:
        - Salvare le spese in un file (es. .csv o .txt) e ricaricarle all’avvio.
- Generare statistiche semplici (opzionale):
        - Mostrare la categoria con la spesa maggiore;
        - Calcolare la media giornaliera delle spese;
